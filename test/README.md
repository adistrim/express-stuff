### Standards

Request methods:
1. GET - To get data from the server
2. POST - To send data to the server
3. PUT - To update data on the server
4. DELETE - To delete data from the server

Status codes:
1. 200 - OK
2. 404 - Not Found
3. 500 - Internal Server Error
4. 441 - Invalid Request
5. 403 - Forbidden
